CLIProxyAPI is the core file of this project. After downloading it from the GitHub repository, you can simply extract it locally and use it directly.

Latest release page: https://github.com/router-for-me/CLIProxyAPI/releases/latest

After extraction, you will find the cli-proxy-api binary file and the config.example.yaml configuration template.

You need to copy config.example.yaml to config.yaml and modify the DIY configuration before running it. Of course, the default configuration can also run directly.
